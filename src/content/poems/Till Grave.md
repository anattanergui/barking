# Till Grave
Rock — is
Without knowing 

That it is a rock
So — is, a cat
Foetus, newborn, asleep
Brain damaged
Cadaver

To be — is 
To be

That
Is the only given
Unknowing, un-being
Is 
Rock, chair, cadaver
Cat

To be — is

Is not 
The same as to be — aware
Nor
The same as being

Being refers to — being aware
That 
Is not a given

To cat, brain damaged
Newborn, asleep, the dead

Being, being aware
Of 
Is the only awareness 

That is — the fall
Human condition 
Split — and 

The drive
Behind all reaching, religions
Devotions, practices
Love and longing

Till grave

On this side 
Of the grave 
Awareness permits 
No experience
No abiding state

But only a concept — 
Of nondual awareness
Unity consciousness
Unsplit duality
Or freedom
