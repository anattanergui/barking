# Doer
There has never been
Never will be
Never can be —

An unshakeable 
Unimpeachable
Reason

To do
Or not do
Anything

Because 
Doing or not doing
Presupposes
A doer

The doer, like an anchor
Seeks a reason —

To do 
Or not do

And finds reasons, like clouds
Drifting in the sky

The doer reaches up
Attempting to grasp
These reasons —

But as it touches the cloud
It passes right through 
With nothing to hold

Not because the clouds
Of reasons aren’t real

But because
The doer itself

Is but a cloud
Shaped like an anchor
