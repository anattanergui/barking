# The Scar
Navel gazing — 
Is that not
The only practice 
If any practice exists
To return

To seek, to find
To discover —
Is to gaze at the scar
Of a wound
You do not remember
Did not cause —

The wound that plunged you 
Into this mess —
You did not create
Did not ask for

Is that not
The way back —
Through that scar

What lies below —
If not
Guts

Is that not 
What anyone is seeking —

Guts
