# Gateway
Philosophy
Is the name
For all things
Related to thought

Reality
Is the name
For all things
Related to senses

When we start thinking
About reality —
That can only be sensed —
We become philosophers 

The basis of philosophy
Is thought
The basis of reality
Is sense

But if thinking itself
Is an object
Perceived by mind —

Then philosophy becomes
The only mechanism
By which we can say
Anything
About reality
