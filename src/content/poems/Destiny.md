# Destiny
One can conceive
Imagine, articulate
Hope, or wish

To have 
What someone else 
Has

Their fortune
Physique, habits
Influence

But one can never
Quite imagine, conceive
Or articulate

The way, in which 
How —  one would be
Different

If one were to have
All that was hoped
And wished

Destiny —
Is not what happens
Nor it is that which is in store
It is 

That, in one —
Which would 
Not be
Different 

If one were to have
All the wished
And hoped
