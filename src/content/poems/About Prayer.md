# About Prayer
Could it be anything
But an attitude

If not
Then it can only be —
Words, postures
Repetitive behaviour

And what is attitude
But a filter —
Through which 
Everything is seen
Touched
And heard

A skeptical, critical
Complimentary, caring, compassionate
Analytical, playful, mournful — 
Attitude

So many attitudes —
Filters
Automatically shifting
Subtly colouring

Everything seen
Heard 
And done

What is prayer then —
Another filter
Or an absence
Of all
