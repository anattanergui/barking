# Misnaming Care
A lamp, an inert object
Does not care
For a table
Another object — thing

Caring requires 
Being — subject
An un-thing
That cares for 
A thing, an object

Thing cannot care for
Un-thing
And un-thing cannot require
Caring

Caring for another 
Then only means
Subject 
Tending to 
An object

Mowing the lawn
Servicing a car
Is care 
Or maintenance 

Between conscious humans
Both subjects —
Care can 
Only be 

Where one is an object 
And another subject

No different from
Maintenance
Servicing the car

Misnaming it care
