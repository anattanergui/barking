# No-Self Talk
All talk 
Is no-self talk

Why

Because there is no one out there
No one else —
And, more importantly
There is no one here either

There is only the talk —
Not a self talking to its own self
But talk talking to talk

Just talking
No-self talk

Who is there 
To talk
To speak
To listen

There is just
Talking
Listening
Speaking — 

And no one there
Who ever spoke
Listened 
Or heard
