# Expression
We write not because 
We know something 
We write because only through it —
We come to know 

Unstated knowing —
Is like an unborn foetus 
It exists —
But neither for itself
Nor for anyone else

Creation, by definition — 
Is expression
Not because we are creative
But because —

Only through expression
We become it —
And only in expression 
Are we creation
