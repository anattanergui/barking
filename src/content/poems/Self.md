# Self
There is not
Nor can be
A neutral
Lens

Through which
Seeing an object
Objectively
Would be 

Objectively
The object is 
That which 
No one sees

Self —
Without lens

Itself
Standing

On either side
Is left 

To be
The only object
Seen 
Objectively

Is objectivity
