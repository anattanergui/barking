# Negation
No system is an open system
A system by definition —
Is closed

There is no possibility, therefore
For any component of a closed system
To be any other way
Than the way it is

Within a system 
Everything is in a closed loop

The conception of openness
Beyond the system’s boundaries
Can only mean one thing —

It must exist simultaneously  — 
Within and alongside — for no outside the system can be imagined —
For the closedness of system 
To ever be comprehended

In which case, it would be identical to the system
Openness identical to closedness
Both negated
